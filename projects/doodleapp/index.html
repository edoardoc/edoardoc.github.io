The shared canvas workflow

  1. Connect both devices to the same local network or hotspot.
  2. Open DoodleTag and select Start Shared Canvas.
  3. Choose your room name and pair with a partner via QR code or PIN.
  4. Sketch together with synchronized layers, colors, and brush settings.
